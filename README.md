## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/MeeturiAjay/RapidReadNews.git
   ```
2. Navigate to the project directory:
   ```sh
   cd RapidReadNews
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the application:
   ```sh
   flutter run
   ```
